## Quick Start (Local or VPS)
1. Install Node.js 18+ and Git.
2. Clone and install:

```bash
git clone https://github.com/Fellobah/felix-md-tech.git
cd felix-md-tech
npm install
node index.js
```

Notes: VPS/Panel/Docker recommended for production. Heroku/Railway/Render are unsupported.
